Overview

The Red Storm Tools Suite is a collection of web-based utilities designed to enhance your Red Storm gameplay experience. All tools run entirely in your browser - no installation required!

What's Included

  • Order of Battle Generator: Generate random aircraft flights using the official OOB tables
  • Chit Pull Generator: Manage initiative and activation chits for gameplay
  • Manual Flight Generator: Create custom flight sheets for any aircraft
  • Aircraft Notes Reference: Quick lookup for special aircraft capabilities and notes
Note: All tools work offline after initial loading. Your data stays in your browser - nothing is sent to a server.

Order of Battle Generator

The OOB Generator allows you to randomly generate aircraft flights using the tables from the Red Storm rulebook.

Getting Started

  1. Select a scenario date (Pre-war or Post-war) - this affects available aircraft
  2. Choose a table from the NATO or Warsaw Pact buttons (A through L)
  3. For tables that require parameters (ATAF zone, nationality, mission type), select from the dropdown
  4. Enter the number of rolls (1-20) you want to make
  5. Click Make Rolls to generate results

Table Types

  • Tables A, B, C: Require ATAF zone selection (2ATAF or 4ATAF)
  • Tables E, K: Require nationality selection for carrier/naval operations
  • Tables F, L: Require mission type selection
  • Tables D, G, H, I, J: No parameters needed, just select and roll

Working with Results

  • Review Results: All generated flights appear in the results panel on the right
  • Generate Flight Sheets: Click "Generate Flight Sheets" to create printable cards for all results
  • Clear Results: Use the "Clear Results" button to start fresh
  • Export: Flight sheets can be printed directly from your browser

Flight Sheet Features

  • Complete aircraft statistics and performance data
  • Weapon loadouts with ammunition tracking
  • Fuel tracking with bubble markers
  • Damage tracking for each aircraft
  • Speed tables for all altitude bands
  • Radar, RWR, and jamming capabilities
Tip: CSAR helicopter flights get special compact cards that are 1/4 the size of regular aircraft cards, allowing up to 4 CSAR flights per row.

Chit Pull Generator

Manage initiative and activation chits for Red Storm gameplay sessions.

How to Use

  1. Configure your chit pools for both NATO and Warsaw Pact forces
  2. Add chits by clicking the "+" buttons for each type
  3. Click Draw Chit to randomly select from the available pool
  4. The drawn chit will be displayed prominently
  5. Use Reset Pool to start a new turn

Chit Types

  • Initiative Chits: Determine turn order
  • Activation Chits: Control when formations activate
  • Special Chits: Weather, events, and special actions
Note: The chit pull tool maintains separate pools for NATO and Warsaw Pact forces, and tracks which chits have been drawn each turn.

Manual Flight Sheet Generator

Create custom flight sheets for specific aircraft without using the random OOB tables.

Creating a Flight Sheet

  1. Select a nation from the dropdown (or choose "All Nations")
  2. Choose an aircraft type from the filtered list
  3. Select flight size (1-ship, 2-ship, or 4-ship)
  4. Click Add to Queue to stage the flight
  5. Repeat for additional flights
  6. Click Print All Flights when ready

Queue Management

  • Add multiple flights to the queue before printing
  • Remove individual flights with the "Remove" button
  • Clear the entire queue with "Clear All"
  • Preview shows the current flight configuration
Tip: Use this tool to create flight sheets for specific scenarios or to replace losses during a campaign.

Aircraft Notes Reference

Quick reference guide for special aircraft capabilities and note codes.

Note Categories

  • Weapon Notes: Special ammunition or weapon system modifications
  • Performance Notes: Speed, altitude, or maneuverability restrictions
  • Capability Notes: Special equipment or operational capabilities
  • Limitation Notes: Operational restrictions or deployment limitations

Using the Reference

  • Browse notes by nation/alliance
  • Search for specific note codes
  • View detailed explanations and game effects
  • See which aircraft are affected by each note

Tips & Best Practices

General Tips

  • Bookmark Tools: Add frequently used tools to your browser bookmarks for quick access
  • Print Settings: Use "Print to PDF" to save flight sheets digitally
  • Browser Compatibility: Best experience in Chrome, Firefox, or Edge
  • Mobile Use: All tools work on tablets, though printing is best on desktop

Flight Sheet Tips

  • Print flight sheets before game sessions to save time
  • Use card sleeves to protect printed sheets for reuse
  • Mark tracking boxes with dry-erase markers if laminated
  • Keep blank sheets on hand for scenario-specific aircraft

OOB Generator Tips

  • Generate all flights for a scenario at once, then print
  • Use the debug view to understand dice rolls
  • Clear results between different scenarios to avoid confusion
  • Note the scenario date setting affects available aircraft
Troubleshooting: If tools aren't working properly, try refreshing the page or clearing your browser cache. All tools work offline after initial loading.
